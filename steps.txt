Steps:

Git
1. Initialize Git in Terminal(local system where you have your repository)
2. Git init the folder
3. Git push the folder to Github

~~Git commands

~ git branch
(to check if you a branch)

~ git checkout -b main
(creating a branch name main0)

~ git add **
(add filenames **)

~ git commit 
(to commit the added files)

~ git push origin main
(to push the committed files to main branch which is in GitHub (remote))

~ git remote -v
(to verify remote repository)

if not set use :
